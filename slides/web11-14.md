引数 `ids` があった時は、フィルタがかかるように

```javascript
// libs/itemsService.js
getItems(ids) {
  // レコード ID が指定されていた場合は、そのレコード ID のみ取得する。
  let params = '';
  if (ids) {
    const query = encodeURIComponent(`$id in (${ids.join(',')})`);
    params += `&query=${query}`;
  }

const uri = `${kintoneApp.base}records.json?app=${kintoneApp.id}${params}`;
debug(`GET ${uri}`);
```
